# Getting Started

This guide will help you get started with `faxai`.

## Installation

### From PyPI

```bash
pip install faxai
```

### From Source

```bash
git clone https://github.com/jparisu/faxai.git
cd faxai
pip install -e .
```

## Requirements

- Python 3.9 or higher
- Dependencies as listed in `pyproject.toml`
