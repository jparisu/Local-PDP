# Getting Started

This guide will help you get started with `lpdp`.

## Installation

### From PyPI

```bash
pip install lpdp
```

### From Source

```bash
git clone https://github.com/jparisu/Local-PDP.git
cd Local-PDP
pip install -e .
```

## Requirements

- Python 3.9 or higher
- Dependencies as listed in `pyproject.toml`
